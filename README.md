
# Ezstock front-end




## Padrão de pr

Mensagem padronizada de pr

```bash
    # <tipo>: <resumo breve da mudança>

    ## Contexto
    Uma breve explicação do problema ou da necessidade que levou à criação do PR.

    ## Mudanças
    - Detalhe 1 das alterações feitas
    - Detalhe 2 das alterações feitas
    - Detalhe 3 das alterações feitas

    ## Como testar
    1. Passo 1 para testar a mudança
    2. Passo 2 para testar a mudança
    3. Passo 3 para testar a mudança
    4. Verifique se todos os testes automatizados passam (`comando para rodar testes`)

    ## Checklist
    - [ ] Código revisado
    - [ ] Testes adicionados
    - [ ] Documentação atualizada
    - [ ] Testes automatizados passam

    ## Referências
    - Fixes #<número da issue>
    - Related to #<número da issue relacionada>

```
